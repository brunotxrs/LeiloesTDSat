# 🏷️ Leilões TDSat

## 📌 Sobre o Projeto
Sistema desktop para gerenciamento de leilões, permitindo cadastro de produtos, registro de lances e consulta de vendas.

## 🛠️ Tecnologias Utilizadas
- Java (JDK 18+)
- MySQL
- Swing (Interface Gráfica)
- Git/GitHub (Versionamento)

## 📋 Funcionalidades
- Cadastro de produtos
- Listagem de produtos
- Registro de vendas
- Controle de status (Vendido/A Venda)

## 🚀 Como Executar
1. Importe o banco de dados `uc11.sql` no MySQL
2. Abra o projeto no NetBeans
3. Execute a classe `cadastroVIEW.java`
